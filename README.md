# 一.功能预览

为了提高模板的通用性与精简性，只实现一些基础的功能：

- 登录 \ 注册
- 密码修改 \ 重置
- 用户管理（增删改分页）
- 个人信息查看 \ 修改
- 头像上传

# 二.必备配置

上述预览效果必要配置项：

- 前端
  - 使用如下指令安装依赖包即可 

```shell
# 1. 安装依赖包
pnpm install
# 2. 启动项目
pnpm dev
```
